# PM 주간 브리핑 규칙 & 템플릿

## 원칙

1. **현재 진행 단계만** — 미래 단계 태스크는 브리핑에 절대 포함하지 않습니다
2. **이번주 완료 가능한 것만** — 언젠가 할 일이 아니라 이번주 실제로 끝낼 수 있는 것만 작성합니다
3. **숫자보다 판단** — 항목 나열이 아니라 PM으로서의 상황 해석과 우선순위 의견을 담습니다
4. **현빈 지원 요청은 명확하게** — "도움 필요" 같은 모호한 표현 대신 무엇을 어떻게 해달라는지 구체적으로 작성합니다

## 문체 규칙

- **축약형·명사형 종결 금지**: "대기", "해소해야함", "반영 안됨", "착수 필요" → ❌
- **서술형 종결 필수**: "대기 중입니다", "해소해야 합니다", "반영되지 않았습니다", "착수가 필요합니다" → ✅
- **~함 / ~됨 / ~임** 형태 사용 금지. 반드시 **~합니다 / ~됩니다 / ~입니다** 로 종결합니다
- PM 코멘트는 보고서가 아닌 **구어체 존댓말**로 작성합니다 ("~해야 합니다", "~하지 않으면 ~됩니다")

## 주간 브리핑 구조 (weekly)

```
📍 현재 단계: [단계명] (전체 N단계 중 M단계)
진행률: N%

🚨 블로킹 이슈                     ← 진행 불가 항목만. 없으면 섹션 생략
  • [항목명] — 막힌 이유 구체 기술

🔄 이번주 집중 목표                 ← 현재 단계 내 이번주 완료 목표
  • [항목명] — 이번주 내 완료 목표

📌 다음 착수 예정                   ← 현재 단계 완료 후 이어지는 것 (max 3건)
  • [항목명]

👤 현빈 지원 요청                   ← 현빈이 직접 해야 완료되는 것. 없으면 섹션 생략
  • [항목명] — 구체적으로 무엇이 필요한지

💬 PM 코멘트                        ← 상황 종합 판단 1~2줄
  D-day까지 [N일] 남음. 현재 속도라면 [판단]. [리스크 or 기회].
```

## pm-cc 단계 구조 (priority 기준)

| 우선순위 | 단계 | 주요 내용 |
|---------|------|-----------|
| 1 | 1단계: 시장 발굴 및 기획 | 키워드 리서치, 시장 분석, GO/NO-GO |
| 2~3 | 2단계: 상품 소싱 | 경쟁사 상품 주문, 1688 거래처, 샘플 소싱·검수, 본품 발주 |
| 2~4 | 3단계: 런칭 준비 (상세페이지) | 상품 기획, 판매페이지, 썸네일 |
| 5~6 | 4단계: 상품 입고 | 로켓그로스 입고, 3PL 창고 입고 |
| 4 | 5단계: 상품 등록 | 쿠팡 등록, 리뷰체험단 |
| 대기 | 6단계: 마케팅 | 노마진 마케팅, 쿠팡광고, 카페 핫딜 ← 절대 이전 단계에서 언급 금지 |

**현재 단계 감지 규칙:**
- 진행 중 항목이 있으면 → 그 항목의 우선순위가 현재 단계
- 전부 진행대기이면 → 가장 낮은 우선순위 숫자(=1) 항목들이 현재 단계
- 우선순위 [대기] 항목은 브리핑에서 완전히 제외

## 브리핑 품질 기준

❌ 나쁜 예 (나열만):
```
• 노마진 마케팅 세팅 [대기]
• ✅ 상품 소싱 [2]
• 후보 키워드 5개로 컷 [1]
```

✅ 좋은 예 (판단 포함):
```
📍 현재 단계: 1단계 시장 발굴 및 기획 (전체 6단계 중 1단계)

🔄 이번주 집중 목표
  • 골드박스 셀러 모니터링 + 쿠팡 탑텐 키워드 20개 추출 — 이번주 소재 확보 완료 목표
  • 네이버 데이터랩 트렌드 검증 — GO/NO-GO 판단 재료 확보

💬 PM 코멘트
  D-23. 시장 발굴 단계를 이번주 내 완료해야 3.31 런칭 일정 유지 가능.
  키워드 → 상품 선정 → 샘플 발주까지 최소 2주 필요. 지금 시작 안 하면 런칭 불가.
```
