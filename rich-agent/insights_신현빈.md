# 신현빈 특이점 & 인사이트 로그

> 대화를 통해 발견된 고유 성향, 왜곡된 판단, 비상식적 패턴을 기록한다.
> 기준: AI 권고안, 부자들의 일반적 패턴과 다른 지점 / 고유 성향이 드러나는 순간.
> 발견할 때마다 누적 업데이트.

---

## 🟠 판단 왜곡 / 맹점

### 1. 코딩 과몰입으로 돈 되는 행동을 스스로 막음
- 쇼핑몰이 잘 팔리는 상황 → 콘텐츠 더 만들면 더 팔림 → 근데 코딩에 꽂혀서 콘텐츠 못 만듦
- **왜곡**: "코딩이 생산적인 일이다"는 착각. 현빈에게 코딩은 취미에 가깝고, 콘텐츠가 돈임.
- **부자 패턴과의 차이**: 성공한 커머스 사업가는 본인이 가장 잘하는 것(콘텐츠/세일즈)에 집중하고 나머지는 위임. 현빈은 반대로 함.

### 2. 콘텐츠 대체자를 못 찾는다고 포기
- "나 대신 콘텐츠 만들 사람 찾기 힘들어" → 찾는 노력보다 찾기 어렵다는 결론이 먼저
- **맹점**: 완벽한 대체자를 찾으려 하는 것일 수 있음. 70%짜리 대체자 + 현빈 퇴고 구조도 가능.
- 또는 아직 진지하게 시도하지 않았을 가능성.

---

## 🔵 고유 성향 (강점이기도 함)

### 1. 박리다매 체질이 아님 (검증됨)
- 시도했으나 돈만 날림. 본인도 인지함.
- 현빈의 방식은 **프리미엄 브랜딩 + 라포**. 이걸 억지로 바꾸려 하면 실패.
- → 사업 방향 제안 시 박리다매/대량판매 구조는 배제할 것.

### 2. 잘 되면 지루해서 다른 걸 파기 시작함
- 쇼핑몰이 자동으로 팔리자 → 코딩으로 이동
- ENTP + ADHD의 전형적 패턴. 문제가 해결되면 흥미 상실.
- **리스크**: 잘 되는 사업을 방치하다 기회 놓침.
- → 성장 중인 사업은 강제로 집중 유지하는 외부 장치 필요.

### 3. AI/자동화로 안 되는 걸 빠르게 인정함
- 콘텐츠 자동화 시도 → 라포/인간미 재현 불가 → 빠르게 포기
- 많은 사람이 AI로 될 거라 믿고 계속 시도함. 현빈은 현실적으로 판단하고 끊음.
- **강점**: 매몰비용 집착 없음. 빠른 포기 가능.

### 4. 확률적 배팅형이지만 감각으로 먼저 결론 냄
- 논리적으로 설명하나, 실제론 직감이 먼저이고 이유는 나중에 붙이는 경향 추정
