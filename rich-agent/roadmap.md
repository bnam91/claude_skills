# 신현빈 1000억 로드맵

> 마감: 2036년 3월 4일 | 작성: 2026년 3월 4일

---

## 핵심 전략 방향

**콘텐츠 → 라포 → IP → 자산**

현빈의 핵심 해자는 **라포 기반 콘텐츠**. AI도 타인도 복제 불가.
- 콘텐츠가 마케팅 비용 0으로 매출을 만드는 구조 → 이걸 레버리지
- 현빈 시간 = 콘텐츠에만 집중. 운영/자동화는 코딩과 팀이 처리
- 콘텐츠 → 쇼핑몰 → 유튜브 → 강의/IP → 투자 순으로 확장
- 박리다매, AI 콘텐츠 자동화는 현빈에게 맞지 않음. 하지 말 것

---

## 10년 로드맵

### Phase 1 — 기반 강화 (2026~2027, 2년)

**목표 자산: 5억 ~ 10억**

| 액션 | 내용 |
|------|------|
| 대행 매출 안정화 | 팔도 관계 고도화. 대형 클라이언트 1~2개 추가 |
| 쇼핑몰 자체 브랜드화 | 사입 → 자체 브랜드 전환. 마진 구조 개선 |
| 유튜브 채널 본격 가동 | AI/바이브코딩 채널 런칭 또는 기존 채널 성장 |
| 콘텐츠 볼륨 확대 | 현빈이 콘텐츠에 쓰는 시간 확보. 쇼핑몰 운영은 자동화/위임으로 분리 |
| 운영 자동화 | 배포·재고·주문 자동화. 바이브코딩은 여기에만 투입 |
| ADHD 루틴 정착 | 타임박스, 알림, 위임 시스템 구축 |

---

### Phase 2 — 스케일업 (2028~2030, 3년)

**목표 자산: 30억 ~ 80억**

| 액션 | 내용 |
|------|------|
| SaaS 제품 런칭 | 마케팅+자동화 결합 SaaS. B2B 월정액 모델 |
| 유튜브 수익화 극대화 | 조회수 → 강의/컨설팅/제품 연계 |
| 대행 → 컨설팅 전환 | 시간 단가 상승. 프리미엄 포지셔닝 |
| 팀 시스템화 | 현빈 없이도 돌아가는 구조. 핵심 인력 충원 |
| 부동산/투자 첫 진입 | 현금흐름 일부를 자산으로 전환 |

---

### Phase 3 — 레버리지 (2031~2033, 3년)

**목표 자산: 150억 ~ 400억**

| 액션 | 내용 |
|------|------|
| SaaS 고객사 확장 | 국내 → 동남아/글로벌 진출 |
| IP 수익 극대화 | 책, 강의, 브랜드 라이선싱 |
| 투자 포트폴리오 구성 | 부동산, 스타트업 지분, ETF 복리 |
| 강연/IR | 현빈의 스토리 자체가 브랜드. 강연·IR 수익 |

---

### Phase 4 — 복리 수확 (2034~2036, 2년)

**목표 자산: 500억 ~ 1000억+**

| 액션 | 내용 |
|------|------|
| SaaS Exit or IPO | M&A 또는 상장. 대형 유동성 이벤트 |
| 투자 수익 본격화 | 복리 자산이 스스로 증식하는 단계 |
| 브랜드 가치 현금화 | IP, 지분, 로열티 |

## 수익원 포트폴리오 목표

| 구분 | 2027 | 2030 | 2036 |
|------|------|------|------|
| 대행/컨설팅 | 50% | 20% | 5% |
| 쇼핑몰/커머스 | 30% | 20% | 10% |
| SaaS/디지털 | 10% | 40% | 30% |
| 유튜브/IP | 10% | 15% | 20% |
| 투자/자산 | 0% | 5% | 35% |

## 핵심 원칙

1. **현빈의 시간 = 콘텐츠에만** — 콘텐츠가 유일한 해자. 코딩이 이 시간을 뺏으면 안 됨
2. **바이브코딩은 운영 자동화에** — 콘텐츠 생산 자동화 시도 금지 (검증됨)
3. **완성 > 완벽** — 출시 후 개선
4. **현금흐름 먼저, 자산은 그 다음** — 수익 없이 투자 없다
5. **레버리지** — 돈이 돈을 벌고, 사람이 사람을 키우는 구조
6. **집중** — 동시에 10개보다 한 번에 1개를 끝까지

## 분기별 체크포인트

rich-agent는 매 분기마다 아래를 점검한다:
- 현재 총 자산 / 목표 대비 진행률
- 수익원별 비중 변화
- 이번 분기 가장 큰 레버리지 포인트
- 다음 분기 집중 액션 1가지
